# Markdown #

### 标题 ###
生成h1到h6标签，是通过在文字前面加上同等个数#符号来实现改变标题字号的大小。出于美观，可以使用对称的闭合式标题符号。
例如：
### 这是标题 ###
# 这是标题 #

这是一个普通段落。
<table>
  <tr>
      <td>Foo</td>
  <tr>
</table>
这是另一个普通段落。

### 字体 ###
* 使用星号和底号'_'表示<em>标签
  例如：*斜体* _斜体_

* 使用双星号和双底号表示<strong>标签
  例如：**强调** __强调__
  
  
### 换行 ###
单一段落用空白行。
