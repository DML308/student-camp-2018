- 1.git add  把工作区的修改，提交到暂存区  git add --all  一次全提交

- 2.git commit 把暂存区的修改，保存到当前分支

- 3.git push  把本地库的记录，推送到远程库

- 4.git status 查看库的状态

- 5.git log 查看日志

- 6.git clone 克隆远程库

- 7.关于工作区暂存区本地仓库远程仓库（顿号怎么打）
